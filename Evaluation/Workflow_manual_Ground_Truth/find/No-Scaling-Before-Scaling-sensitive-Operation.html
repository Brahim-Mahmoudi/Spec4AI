🔎 Exhaustive Checking Steps for No Scaling Before Scaling-sensitive Operation

  1. Check if scikit-learn (sklearn) is imported.

    • import sklearn

    • or from sklearn import ...

  2. Search for usage of scaling-sensitive operations:

    • PCA()

    • SVC()

    • SGDClassifier()

    • SGDRegressor()

    • MLPClassifier()

    • MLPRegressor()

    • Ridge()

    • Lasso()

    • ElasticNet()

  3. For each of these:

    • Check if a scaler (StandardScaler(), MinMaxScaler(), RobustScaler(), Normalizer(), etc.) is applied before the sensitive operation.

    • Typically this is done through a Pipeline() or manual preprocessing.

  4. If no scaling is done before, mark as No Scaling Before Scaling-sensitive Operation.

  5. If scikit-learn is not imported, skip the check.
