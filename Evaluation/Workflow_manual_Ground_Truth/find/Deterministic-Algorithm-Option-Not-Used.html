🔎 Exhaustive Checking Steps for Deterministic Algorithm Option Not Used

  1. Check if any of these libraries is imported:

    • torch (PyTorch)

    • tensorflow

  2. If PyTorch is imported:

    2.1 Before any model training (model.forward(), loss.backward(), optimizer.step(), or any training loop):

    • Search if:

      • torch.use_deterministic_algorithms(True) is called.

    ✅ If not set to True before training starts, mark as Deterministic Algorithm Option Not Used.

  3. If TensorFlow is imported:

    3.1 Before any model training (model.fit(), GradientTape manual training loops, etc.):

    • Search if:

      • tf.config.experimental.enable_op_determinism() is called.

    ✅ If not called before training, mark as Deterministic Algorithm Option Not Used.

  4. If none of these libraries are imported:

  • Skip this check.
