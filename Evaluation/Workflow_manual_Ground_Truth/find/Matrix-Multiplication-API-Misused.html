🔎 Exhaustive Checking Steps for Matrix Multiplication API Misused

2D matrix -> a = [[1, 0], [0, 1]]

  1. Check if numpy is imported.

    • import numpy as np

    • from numpy import ...

  2. If NumPy is imported:

    • Search for any usage of np.dot().

  3. When np.dot() is used:

    • Check if both operands are two-dimensional arrays (ndim == 2).

  4. If both operands are 2D, mark as Matrix Multiplication API Misused.

  5. Expected alternative: use np.matmul() instead for 2D matrix multiplication.

  6. If NumPy is not imported, skip checking.
